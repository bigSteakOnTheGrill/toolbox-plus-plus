Needed Software

For the compilation of the chm help the software 'HTML Help Workshop and Documentation' has to be installed (e.g. from https://www.microsoft.com/en-us/download/details.aspx?id=21138).

The workshop software should be installed at C:\Program Files (x86)\HTML Help Workshop. If another location is used, the file compileHelp.bat has to be modified.

For the editing the HTML help files any HTML editor can be used. For example Microsoft Expression Web 4 can be downloaded and used for free.
