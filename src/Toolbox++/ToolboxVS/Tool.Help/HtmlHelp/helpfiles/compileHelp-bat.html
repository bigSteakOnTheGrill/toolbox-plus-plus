compileHelp.bat

The script compiles the chm file and triggers the generation of index and include files for VBA, C++ and C.

Each html file gets an unique index. The indexing starts with 1000 and increase by 10 for each html file. Furthermore each file gets an unique key. With the key the help ID can be found.

The following index and include files are generated:

  • Toolbox++ChmHelpHelpIDsIndex.txt
    which contains a text searchable keyword and the corresponding help ID.

  • Toolbox++ChmHelpHelpIDsInclude.h
    which contains an enum list with the help IDs. This can be used like this:

    namespace eChmHelpID {
        enum Enum {
            #include <Toolbox++ChmHelpHelpIDsInclude.h> // this file will be generated by Tool.Help.
            enUnknown = 0
        };
    }
  • Toolbox++ChmHelpHelpIDsInclude.cpp
    which can be used a text search in C or C++. For example:

    typedef struct {
        eChmHelpID::Enum helpID;
        const wchar_t *key;
    } HELPID_INFO_STRUCT;
    static const HELPID_INFO_STRUCT myHelpIDInfo[];

    const HELPID_INFO_STRUCT myHelpIDInfo[] = {
        #include <Toolbox++ChmHelpHelpIDsInclude.cpp> // This file will be generated by Tool.Help.
        {eChmHelpID::enUnknown , L"Unknown"}
    };
